# Challenge Excape website

Kleine statische website voor GitHub of eenvoudige hosting.

## Structuur

- `index.html` — volledige pagina
- `styles.css` — styling
- `script.js` — mobiel menu
- `assets/` — gebruikte afbeeldingen

## Lokaal openen

Open `index.html` direct in je browser.

## Naar GitHub zetten

1. Maak een nieuwe repository aan.
2. Upload alle bestanden uit deze map.
3. Commit en push.
4. Activeer eventueel GitHub Pages op de branch met deze bestanden.

## Aanpassen

- Teksten staan direct in `index.html`
- Kleuren en spacing staan in `styles.css`
- Afbeeldingen vervang je in `assets/`
